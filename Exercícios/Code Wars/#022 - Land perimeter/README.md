# Descrição
Dado um array arr de strings, complete a função calculando o perímetro total de todas as ilhas. Cada pedaço de terra será marcado com 'X' enquanto os campos de água são representados como 'O'. Considere cada tile sendo um pedaço de terra perfeito de 1 x 1. Alguns exemplos para melhor visualização:

['XOOXO',

'XOOXO',

'OOOXO',

'XXOXO',

'OXOOO']

deve retornar: "Perímetro total da terra: 24".

Seguinte entrada:

['XOOO',

'XOXO',

'XOXO',

'OOXX',

'OOOO']

deve retornar: "Perímetro total da terra: 18"
